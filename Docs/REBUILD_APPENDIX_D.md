## D.5 FFT 测量 — 基本值（阶段 4）

> **验收方式**：本阶段以 **Keil Watch** 为准，**不要求** OLED L3～L7 全参显示（属 **D.11 / 阶段 6**）。  
> 上电后 OLED 仅显示启动屏（`Stage4 FFT`）不影响 D.5 通过判定。

### 操作步骤

1. 烧录后 **Run** 约 1 s（等待 4 帧滑动平均收敛）
2. **Halt** 调试暂停
3. 在 **Watch** 中添加下列表达式

### Watch 必看项与通过标准

| Watch 表达式 | 通过标准 |
|--------------|----------|
| `g_AppParams.mode` | `0`（手动） |
| `g_AppParams.target_freq_hz` | `1000` |
| `g_MeasureResult.ui_rms_v` | **0.080～0.120**（V） |
| `g_MeasureResult.uo_rms_v` | 与运放增益一致，非 0 |
| `g_MeasureResult.av_gain` | ≈ `uo_rms_v / ui_rms_v`；反相板为**负** |
| `g_MeasureResult.phase_deg` | **±160°～±180°**（±20° 噪声可接受） |
| `g_MeasureResult.meas_freq_hz` | **999.5～1000.5** |
| `g_MeasureResult.freq_ok` | `1` |
| `g_MeasureResult.fft_fail_code` | `0` |
| `g_FFT_DbgTargetBin` | **20**（1 kHz @ ≈51.2 kSa/s） |
| `g_FFT_DbgPeakBin` | **18～22** |
| `g_FFT_DbgUiPeakMag` | **> 40**（自动增益后浮点幅值量纲，峰值标尺 ≈28000） |
| `g_MeasureResult.adc_ui_max - g_MeasureResult.adc_ui_min` | **≥ 250**（码值峰峰值，约对应 80 mV 级以上交流） |

### fail_code 速查

| 值 | 含义 | 常见原因 |
|----|------|----------|
| 0 | 通过 | — |
| 1 | Ui ADC 异常 | 缓冲贴 0 或满量程 |
| 2 | Uo ADC 异常 | 同上 |
| 3 | Ui 无有效峰 | 信号过小 / PA4→PA2 链路断 |
| 4 | Uo 无有效峰 | PA3 未接或 Uo 过小 |

### 旁路自检（可选）

将 **PA4 直连 PA2**（断开运放板），Run 1 s → Halt：`fft_fail_code` 应为 **0**，`ui_rms_v` ≈ **0.09～0.11 V**。
