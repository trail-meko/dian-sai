# 反向比例运放增益自动测试系统 — 设计报告

## 1. 系统概述

### 1.1 设计目标

基于 **STM32F407ZGT6**（168 MHz SYSCLK，1 MB Flash / 192 KB RAM，Cortex-M4F）构建运放增益自动测试平台，实现：

1. **可控激励**：DAC 输出 1.0～2.0 kHz 正弦/三角波，经耦合电容驱动被测反向比例运放；
2. **双路同步采集**：扫描采集运放输入 Ui、输出 Uo；
3. **频域解算**：FFT 求 RMS、增益 Av、Ui/Uo 相位差；
4. **自动稳幅**（扩展）：PID 闭环调节 DAC 幅度，使 Uo 有效值稳定在 1.5 V。

系统为 **中断驱动**：主循环空转，DMA 帧完成触发 FFT/PID，TIM4 刷新 OLED，TIM7 扫描按键。

### 1.2 硬件平台

| 项目 | 规格 |
|------|------|
| MCU | STM32F407ZGT6（LQFP144），Cortex-M4F |
| 系统时钟 | 168 MHz（`SYSCLK_HZ`） |
| 定时器时钟 | TIM6/TIM2 = 84 MHz（APB1 定时器 ×2） |
| 信号输出 | 12-bit DAC1（PA4 / TP_DAC） |
| 信号输入 | 12-bit ADC1 扫描（PB0=Ui，PB1=Uo） |
| 显示/交互 | OLED（I2C）+ 3 键 |

### 1.3 信号流

```
波形查表 → Q15 幅度缩放 → TIM6+DMA1 → DAC(PA4)
    → [耦合电容] → 运放(Ui/Uo) → ADC1 扫描采样
    → DMA 缓冲 → 去直流+自动增益 → 1024点 f32 FFT
    → RMS / 增益 / 相位差 → OLED 显示
```

## 4. 信号产生模块

### 4.1 方案选型

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| 外部 DDS 芯片 | 频率精度高、波形丰富 | 额外硬件、接口复杂 | 不采用 |
| PWM + RC 滤波 | 电路简单 | 谐波大、频率范围受限 | 不采用 |
| **MCU DAC + 查表 + DMA** | 集成度高、幅度软件可控、与 PID 闭环兼容 | 频率/波形受内存与定时器约束 | **采用** |

本系统采用 **软件波形查表 + 内部 DAC + TIM6 更新触发 DMA** 方案。F407 以 **TIM6_TRGO + DMA1_Stream5 Ch7** 写 `DAC->DHR12R1` 实现连续波形输出。

### 4.2 工作原理

#### （1）波形生成

- **正弦波 / 三角波**：Flash 预存查表（`DAC_SinTables.inc` / `DAC_TriTables.inc`），峰值 ±2047；
- 支持 N = 242/249/250/251/252/254/256（F407 版新增 N=242、251）。

#### （2）频率控制

频率公式：

```
f = TIM6_CLK_HZ / [(PSC+1) × (ARR+1) × N]
  = 84 MHz / [(PSC+1) × (ARR+1) × N]
```

11 档频率在编译期预计算 `(N, PSC, ARR)` 三元组（见附录 C）。切频流程：DAC 输出中点 → 停止 DMA → 重建波形表 → 重配 TIM/DMA → 重启，避免波形突变。

#### （3）幅度控制

查表值经 **Q15 定点增益** `s_GainScaleQ15` 缩放后偏移至 12-bit 中点 2048：

```
DAC_code = 2048 + (WaveTable[i] × GainScaleQ15) >> 15
```

- **手动模式**：固定 `MANUAL_GAIN_SCALE_Q15 ≈ 2785`，目标 Ui RMS ≈ 100 mV；
- **自动模式**：PID 调节增益，目标 Uo RMS = 1.5 V。

#### （4）DMA 传输配置

| 参数 | 配置 |
|------|------|
| 流/通道 | **DMA1_Stream5**，Channel **7** |
| 方向 | 内存 → 外设（DAC DHR12R1） |
| 数据宽度 | 半字（16 bit） |
| 模式 | 循环（Circular） |
| 触发源 | TIM6 更新事件 |

### 4.3 设计要点

1. **每档独立 N**：在频率精度与谐波抑制之间折中；F407 部分档位 N 相对 F103 微调以满足 0.5‰；
2. **切频无毛刺**：先输出中点电平、停 DMA 再重配；
3. **幅度闭环兼容**：DAC 模块只负责执行 Q15 系数，PID 在 `PID_Ctrl.c` 中调节；
4. **测试点**：TP_DAC（PA4）经耦合电容接运放输入。

## 5. 信号采集模块

### 5.1 方案选型

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| 单 ADC 轮询 | 实现简单 | 两路非同步，相位测量无效 | 不采用 |
| 双 ADC 独立触发 | 硬件简单 | 采样时刻不对齐 | 不采用 |
| **ADC1 扫描模式** | 同触发周期内顺序采 Ui/Uo，配合延迟补偿可做相位 | 需补偿 Rank 间延迟 | **采用** |
| 外部高速 ADC | 采样率高 | 成本高、接口复杂 | 不采用 |

### 5.2 工作原理

#### （1）引脚与通道分配

| 测试点 | 引脚 | ADC 通道 | 信号 |
|--------|------|----------|------|
| TP_Ui | PB0 | ADC1_IN8 | 运放输入端交流信号 |
| TP_Uo | PB1 | ADC1_IN9 | 运放输出端交流信号 |

#### （2）ADC 配置

- **工作模式**：ADC1 独立 + **扫描**（Rank1=CH2，Rank2=CH3）；
- **触发源**：TIM2 CC2 外部触发，每个脉冲转换一对数据；
- **数据对齐**：右对齐 12 bit；
- **采样时间**：480 cycles，ADC 时钟 = 21 MHz。

#### （3）采样率计算

```
Fs = TIM2_CLK_HZ / (PSC+1) / (ARR+1)
   = 84 MHz / 1 / 1639
   ≈ 51.3 kHz
```

相对最高信号频率 2 kHz，满足 Nyquist 准则。一帧 1024 点对应约 20 ms。

#### （4）DMA 搬运

| 参数 | 配置 |
|------|------|
| 流/通道 | **DMA2_Stream0**，Channel **0** |
| 外设地址 | `ADC1->DR` |
| 缓冲区长度 | 2048 半字（拆为 Ui/Uo 各 1024） |
| 模式 | 循环 |
| 中断 | 传输完成 → 拆包 → 置 `g_ADC_FrameReady` |

#### （5）数据完整性保障

- 扫描采样 + 软件延迟补偿保证相位可测；
- DMA 循环缓冲连续覆盖，无丢样；
- 异常检测：ADC 全 0 或全满码时判为无效帧。

### 5.3 设计要点

1. **定时器独立**：TIM2 专用于 ADC 触发，与 DAC 的 TIM6 独立；
2. **整帧触发处理**：1024 点采满后由 TIM7 触发 FFT；
3. **中断优先级**：DMA2 Stream0 设为最高优先级；
4. **模拟前端**：交流耦合隔直流，FFT 前去直流。

## 6. 相位测量方案论证

### 6.1 测量需求

反向比例运放增益测试需获取 **Ui 与 Uo 的相位差 Δφ**：

- 反相放大理论值 Δφ ≈ 180°；
- 相位信息辅助判断增益极性（|Δφ| > 90° 时增益显示为负）；
- 设计指标：|Δφ| 测量误差 ≤ 3°（`PHASE_ERROR_DEG`）。

被测信号为 **单频稳态**（1～2 kHz），适合频域相位法。

### 6.2 选定方案：FFT 单频相位法

#### 6.2.1 算法流程

```
Ui[n], Uo[n]  (各 1024 点)
    ↓ 去直流均值
    ↓ 自动增益至峰值 ≈ 28000.0f
    ↓ arm_cfft_f32（1024 点）
    ↓ 在目标频率 bin ±2 范围内搜峰值
    ↓ φ_Ui = atan2(Im, Re)，φ_Uo 同理
    ↓ Δφ = φ_Uo − φ_Ui，归一化到 (−180°, +180°]
    ↓ 减去扫描延迟校正（ADC_INTERCH_DELAY_SEC）
    ↓ 4 帧滑动平均
    → 输出 g_MeasureResult.phase_deg
```

#### 6.2.2 选型理由

1. **架构一致**：RMS、频率、增益、相位共用一套 FFT 流水线；
2. **F407 FPU**：`arm_cfft_f32` 利用 Cortex-M4F 浮点单元，实时性优于 F103 定点库；
3. **单频稳态匹配**：已知激励频率，可在目标 bin 邻域搜峰；
4. **反向运放判别**：Δφ ≈ 180° 时增益显示为负。

#### 6.2.3 精度保障措施

| 措施 | 作用 |
|------|------|
| 去直流 | 避免 DC 分量干扰低频 bin |
| 自动增益 | 稳定 FFT 动态范围 |
| 目标 bin ±3 搜峰 | 补偿栅栏效应 |
| 扫描延迟补偿 | 修正 ADC1 扫描 Rank 间时序偏差 |
| 4 帧滑动平均 | 降低随机噪声 |
| Fs ≈ 51.2 kHz，N = 1024 | 分辨率 ≈ 50 Hz |

## 7. 小结

| 模块 | 核心方案 | 关键外设 |
|------|----------|----------|
| 信号产生 | 查表 + Q15 幅度 + TIM6/DMA1/DAC | TIM6、DMA1_Stream5、DAC1 |
| 信号采集 | ADC1 扫描 + TIM2 触发 + DMA | TIM2、ADC1、DMA2_Stream0 |
| 相位测量 | FFT 单频 atan2 法 + 4 帧平均 + 延迟补偿 | CMSIS-DSP `arm_cfft_f32` |

三模块形成 **DAC 激励 → 同步采集 → 频域解算** 闭环测量链，满足反向比例运放增益与相位自动化测试需求。
