# 附录 A：硬件接线拓扑与引脚

## A.1 被测对象（定性，不写阻值）

板载 **反相比例运算放大器** 增益测试电路：

```
                    耦合电容（隔直流）
TP_DAC (PA4) ──────────────────────→ 运放输入网络
                                          │
                                     [反相放大器]
                                          │
TP_Ui  (PA5) ←── 探测运放输入端交流信号 ──┤
TP_Uo  (PA6) ←── 探测运放输出端交流信号 ──┘
```

- **不要求**在固件中写入 Rf/Ri 或理论增益；Av 完全由 **Uo_RMS / Ui_RMS** 实测。
- 跳线 **S2 / S3 / S4** 改变外围电阻配置，从而改变实测增益；**软件不读这些 GPIO**。

## A.2 MCU 与引脚表（与 `app_config.h` 一致）

| 测试点/外设 | MCU 引脚 | 说明 |
|-------------|----------|------|
| TP_DAC | PA4 | DAC1 Ch1 模拟输出，12-bit，中点偏置 2048 |
| TP_Ui | PA5 | ADC1_IN5，运放输入端交流信号 |
| TP_Uo | PA6 | ADC1_IN6，运放输出端交流信号 |
| KEY 波形 | PE2 | 正弦/三角波切换，低电平有效，内部上拉 |
| KEY 频率 | PE3 | 频率 +100 Hz，低电平有效 |
| KEY 模式 | PE5 | 手动/自动模式切换 |
| OLED SCL | PC8 | 软件 I2C（F407 上 PC8/9 无硬件 I2C1 AF） |
| OLED SDA | PC9 | 软件 I2C |
| 故障 LED | PF9 | 有故障标志时闪烁，无故障时常灭（低电平点亮） |
| 板载 LED1 | PF10 | 仅 `LED_Init()` 初始化，应用层可不使用 |

OLED I2C 地址：优先 **0x78**（8-bit 写地址），备选 **0x7A**。

## A.3 模拟前端假设

- DAC 输出经 **交流耦合** 后进入运放；ADC 采集的是 **交流分量**（允许存在中点偏置，FFT 前去直流）。
- ADC 参考电压 **Vref = 3.3 V**，12-bit（0～4095）。
- Ui 手动模式目标有效值约 **100 mV**（通过 Flash 标定 `dac_vpp_100mV` 保证）；自动模式 PI 目标 **Uo 有效值 1.5 V**。

## A.4 与 F103 第3版差异（勿照搬）

以下出现在 F103 第3版重建文档或旧设计中，与 **F407 第1版** 不一致，**忽略**：

| 过时描述（F103 第3版） | F407 第1版实际 |
|------------------------|----------------|
| MCU STM32F103RCT6 | **STM32F407ZGT6** |
| Ui/Uo = PA2/PA3 | **PA5/PA6** |
| 按键 PB10/PB11/PB12 | **PE3/PE5/PE2** |
| OLED PB8/PB9 | **PC8/PC9** |
| DAC TIM6 + DMA2_Ch3 | **TIM2 + DMA1_Stream5** |
| ADC TIM2 触发 + DMA1_Ch1 | **TIM3 触发 + DMA2_Stream0** |
| FFT cr4_fft_1024 定点 1024 点 | **arm_cfft_f32 浮点 2048 点 + Hann 窗** |
| TIM7 后台 + 空 while(1) | **SysTick + PendSV + __WFI()** |
| 11 档 N/PSC/ARR 频率表 | **固定 256 点/周期，改 TIM2 ARR** |
| 故障 LED PA8 | **PF9** |
| 采样率 ≈51.2 kHz | **100 kHz** |
